AndI Lua Scripting API

Welcome to the AndI Lua Scripting API. Access physical hardware, local AI models, and UI components synchronously within the Lua Isolate.

Tip: Heavy tasks like llamaGenerate or openCamera delegate work to the Main Dart Isolate to prevent script freezing.

🤖 Artificial Intelligence

Interact with local and cloud-based models.

FunctionDescription
llamaGenerate(prompt) Prompts the local GGUF Llama model. Output streams asynchronously to the Terminal.
geminiGenerate(prompt) Prompts Google Gemini API (requires network).

📷 Camera & Computer Vision

Access the high-performance native Double Buffer.

  • openCamera(index, res): Index 0 for rear, 1 for front. Resolutions: low to max.
  • getCameraFrame(): Non-blocking read. Returns {width, height, format, data}.
  • closeCamera(): Disposes resources.

🧠 ONNX Machine Learning

Load models into the native C-Heap for low-latency inference.

-- Load and run a model
onnxLoad("/path/to/model.onnx") --
onnxRun("/path/to/model.onnx", '{"input": [[0.5]]}') --
onnxDispose("/path/to/model.onnx") --

📟 Hardware Sensors

Read XYZ vectors from Native Shared Memory in O(1) time.

Important: Call initSensors() before use and disposeSensors() after to save battery.
  • getAccelerometer(), getGyroscope(), getMagnetometer().

🚀 Sample Script: Vision Logic

This script combines camera access with UI feedback based on orientation.

initSensors() --
openCamera(0, "low") --

while true do
    local acc = getAccelerometer() --
    if acc[2] > 9.0 then
        setEmotion("happy") -- Phone is upright
    end
    
    local frame = getCameraFrame() --
    if frame ~= nil then
        log("Processed frame: " .. frame.width) --
    end
end
