Integrate device hardware, custom machine learning, and AI chat capabilities into your AndI scripts.

📷 Camera & Computer Vision

Capture frames into a high-performance Double Buffer.

openCamera(0, "medium") -- Open rear cam
local frame = getCameraFrame() -- Non-blocking read
if frame then
    log("Resolution: " .. frame.width .. "x" .. frame.height)
end

🧠 ONNX Machine Learning

Load and run .onnx models directly in the C-Heap for real-time inference.

  • onnxLoad(path): Prepares model memory.
  • onnxRun(path, json): Executes inference.
  • onnxDispose(path): Frees C-Heap memory.

📟 Hardware Sensors

Battery Warning: You must call disposeSensors() when finished to power down the IMU hardware.

Access Accelerometer, Gyroscope, and Magnetometer data in O(1) time.

🚀 Full Integration Example

Combining sensors and UI for an "Orientation-Aware" robot personality:

initSensors() 
while true do
    local acc = getAccelerometer()
    
    -- If phone is tilted up (Y-axis > 9)
    if acc[2] > 9.0 then
        setEmotion("happy")
        showToast("Feeling good!")
    else
        setEmotion("normal")
    end
end
